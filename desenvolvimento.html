INTRODUÇÃO

Nesta seção, você verá as principais tecnologias utilizadas no desenvolvimento do site R.A.P. A imagem abaixo representa visualmente as linguagens e ferramentas que fizeram parte da construção do projeto.

O Motivo de Escolhermos essas Linguagens

HTML

Utilizamos o HTML para estruturar todo o conteúdo do site. Ele é a base da web e nos permite organizar os elementos de forma semântica e acessível.

CSS

O CSS foi essencial para estilizar o site, definir cores, fontes, espaçamentos e criar animações que tornam a experiência mais agradável e moderna.

JavaScript

Com o JavaScript conseguimos adicionar interatividade, como animações, efeitos visuais e respostas dinâmicas às ações do usuário.
